# Executive Function and Decision-Making Analysis

## Overview
This project examines whether executive function measures predict decision-making performance.

## Research Question
Does executive function (reaction time and variability) predict decision-making ability?

## Methods
- Multiple linear regression
- Variables: reaction time, variability, age
- Tool: R

## Results
Reaction time variability significantly predicted decision-making performance, suggesting that greater inconsistency in cognitive processing is associated with poorer outcomes.
